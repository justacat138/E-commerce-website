# E-commerce AI Platform 🚀

Dự án website thương mại điện tử tích hợp AI, được phát triển bằng công nghệ Next.js hiện đại, tập trung vào trải nghiệm người dùng thông minh và quản lý quy trình chuyên nghiệp qua Jira. Dự án sử dụng Framework Cursor AI.

---

## 🛠 Tech Stack (Công nghệ sử dụng)

| Layer | Technology | Chi tiết |
| :--- | :--- | :--- |
| **Frontend** | **React & Next.js** | App Router, UI components. |
| **Language** | **TypeScript** | Sử dụng xuyên suốt cho cả Client và Server. |
| **Styling** | **Tailwind CSS** | Thiết kế giao diện nhanh, responsive. |
| **Backend** | **Next.js Server** | Server Components, Route Handlers (API), Server Actions. |
| **Tools** | **Cursor AI & Jira** | Hỗ trợ code bằng AI và quản lý Task theo chuẩn Agile. |

---

## 📋 Danh sách Use Cases (Quản lý trên Jira)

Mỗi commit code sẽ được ánh xạ trực tiếp vào mã SCRUM tương ứng:

* **SCRUM-4:** Giỏ hàng thông minh (Sắp xếp món mới nhất lên đầu).
* **SCRUM-5:** Tra cứu phí vận chuyển dựa trên vị trí địa lý.
* **SCRUM-6:** Thanh toán đa phương thức (Bank, Momo, COD).
* **SCRUM-7:** Bộ lọc Shop uy tín bằng AI.
* **SCRUM-8:** Hệ thống lọc Review ảo bằng AI.

## 🚀 Hướng dẫn Cài đặt & Chạy dự án (Local)

### 1. Cài đặt thư viện (Dependencies)
Mở Terminal tại thư mục gốc của dự án và chạy:
```bash
npm install
```
### 2. Chạy chế độ Phát triển (Development)
Khởi động server bằng lệnh:
``` bash
npm run dev
```
Mở trình duyệt: http://localhost:3000
### 3. Build & Chạy Production
```Bash
npm run build
npm run start
```
